## Usage

### secrets file

Create a secrets.js file in the src folder

```
export const ARTICLE_SEARCH_API_KEY = YOUR_ARTICLE_SEARCH_API_KEY;
export const POPULAR_ARTICLE_API_KEY = YOUR_POPULAR_ARTICLE_API_KEY;

```

### Install Dependencies

```
npm i
```

### Run web-app(:3232)

```
npm start
```
